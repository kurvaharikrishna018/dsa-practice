# DSA Practice

A comprehensive collection of Data Structures and Algorithms implementations and practice problems. This repository covers fundamental to advanced data structures with detailed explanations, implementations, and examples.

## 📚 Data Structures Covered

### 🔤 **Array**
- Basic operations (insert, delete, search, update)
- Multi-dimensional arrays
- Dynamic arrays
- Array manipulation algorithms
- Time and space complexity analysis

### 🔗 **String**
- String operations and manipulation
- Pattern matching algorithms
- String searching techniques
- Palindrome and anagram checks
- String compression and encoding

### 📋 **List**
- ArrayList vs LinkedList comparison
- List operations and implementations
- Iterator patterns
- List sorting and searching

### 🔗 **Linked List**
- Singly Linked List
- Doubly Linked List
- Circular Linked List
- Operations: insertion, deletion, traversal
- Linked list algorithms (reversal, cycle detection, etc.)

### 🌳 **Tree**
- Binary Tree
- Binary Search Tree (BST)
- AVL Tree
- Red-Black Tree
- B-Tree and B+ Tree
- Tree traversals (Inorder, Preorder, Postorder, Level-order)
- Tree operations and balancing

### 🗺️ **Map**
- HashMap implementation
- TreeMap (ordered map)
- Map operations and use cases
- Collision resolution techniques
- Time complexity analysis

### 🎯 **Set**
- HashSet implementation
- TreeSet (ordered set)
- Set operations and properties
- Set algorithms and applications

### 🌐 **Graph**
- Graph representations (Adjacency Matrix, Adjacency List)
- Graph traversals (BFS, DFS)
- Shortest path algorithms (Dijkstra, Bellman-Ford)
- Minimum spanning tree (Prim's, Kruskal's)
- Topological sorting
- Graph coloring and cycle detection

### 🔑 **Hash Table**
- Hash functions and collision resolution
- Chaining and Open Addressing
- Load factor and rehashing
- Hash table applications and optimizations

4. **Expert**: Advanced Tree Structures, Complex Graph Algorithms, Optimization Techniques
